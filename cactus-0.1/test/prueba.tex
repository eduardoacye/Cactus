La clásica función factorial.
$$n!=n*(n-1)*(n-1)*...*2*1$$

\bigskip


\bigskip

\bigskip

\textbf{factorial :} \emph{numero} $\boldsymbol{\rightarrow}$ \emph{numero}


Return n! = n(n-1)...1 .
\begin{verbatim}
(define (factorial n)
  (if (= n 0)
      1
      (* n (factorial (- n 1)))))
\end{verbatim}
